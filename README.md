<h2>üó∫Ô∏è What is a Normal Map?</h2>

<p>
A <b>Normal Map</b> in Power BI is a map visual that displays geographic locations
using <b>bubbles or points</b> based on latitude, longitude, or location fields.
</p>

<p><b>Key Features:</b></p>

<ul>
  <li>Uses bubble size to represent values</li>
  <li>Supports cities, states, countries, and coordinates</li>
  <li>Ideal for showing sales, profit, or quantity distribution</li>
</ul>

<p>
Example: Showing total sales by city using bubble size.
</p>
<h2>üåç What is a Filled Map?</h2>

<p>
A <b>Filled Map</b> is a geographic visualization where regions such as 
<b>countries, states, or districts</b> are filled with colors based on values.
</p>

<p><b>Key Features:</b></p>

<ul>
  <li>Uses color intensity to represent data</li>
  <li>Best for regional comparison</li>
  <li>Commonly used for performance analysis</li>
</ul>
