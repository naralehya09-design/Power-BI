        <h2>üó∫Ô∏è What is Normal Map?</h2>
        <p>

            <b>bubbles or points</b> on the map.
        </p>

        <p><b>Key Features:</b></p>
        <ul>
            <li>Uses bubbles to represent values</li>
            <li>Supports city, state, country, and latitude/longitude</li>

        </ul>

            Example: Total sales by city represented using bubble size.
        </p>

        <h2>üåç What is Filled Map?</h2>
        <p>
