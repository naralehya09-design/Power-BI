  <h2>ğŸ“Š Bar Charts in Power BI</h2>

  <h3>ğŸ”¹ Stacked & 100% Stacked Charts</h3>

  </ul>

    <li><strong>Note:</strong> Histogram is <strong>not directly available</strong> as a default visual in Power BI.</li>
  </ul>

  <h3>ğŸ”¹ Types of Data Analysis</h3>

    <li><strong>Bivariate Analysis</strong> â€“ Analysis between two variables.</li>
    <li><strong>Multivariate Analysis</strong> â€“ Analysis involving more than two variables.</li>
  </ol>

  <h3>ğŸ”¹ Types of Columns Used</h3>
